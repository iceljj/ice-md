Runtime API Examples ​

This page demonstrates usage of some of the runtime APIs provided by VitePress.

The main useData() API can be used to access site, theme, and page data for the current page. It works in both .md and .vue files:

md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

### Theme Data

### Page Data

🌟 6. Markdown 可视化编辑器（受 Typora / MarkText 启发） ​

不只是语法高亮，而是“所见即所得”的 Markdown 编辑体验。

🎯 功能： ​

  • 实时预览（左侧编辑，右侧渲染，或内联渲染）
  • 支持表格、代码块、数学公式（LaTeX）、流程图（Mermaid）
  • 图片拖拽上传 & 自动插入
  • 主题切换（深色/浅色）、快捷键支持

⚙️ 技术难点： ​

  • 集成 marked.js 或 markdown-it 解析 Markdown
  • 使用 highlight.js 或 Prism.js 实现代码高亮
  • 集成 MathJax 或 KaTeX 渲染数学公式
  • 集成 Mermaid.js 渲染流程图/序列图
  • 拖拽上传图片并调用 API 上传（模拟 GitHub 图床）
  • 使用 contenteditable 实现富文本编辑（进阶）

💡 为什么有趣？ ​

  • 接近现代笔记工具（如 Notion、语雀）
  • 综合前端技术栈：解析、渲染、交互、异步

🌟 7. 类 Excel 在线表格编辑器（受 handsontable / luckysheet 启发） ​

实现一个支持单元格编辑、公式计算、行列操作的表格组件。

🎯 核心功能： ​

  • 单元格点击/双击编辑
  • 支持键盘导航（方向键、Enter、Tab）
  • 支持复制粘贴（Ctrl+C/V）
  • 行列增删、冻结行列
  • 基础公式计算（如 =SUM(A1:A5)）
  • 单元格格式（文本、数字、日期）

⚙️ 技术难点： ​

  • 虚拟滚动（Virtual Scrolling）：处理上千行数据时不卡顿（使用 CDK Scrolling）
  • 二维数据结构管理：用 Map<string, Cell> 或二维数组管理数据（A1 → [0][0]）
  • 编辑模式管理：区分“选择模式”和“编辑模式”
  • 键盘事件监听：全局监听 @HostListener('document:keydown')
  • 公式解析引擎：可用 hot-formula-parser 或自己实现简单解析
  • 变更检测优化：使用 ChangeDetectionStrategy.OnPush

💡 为什么有趣？ ​

  • 接近 Excel 的交互体验
  • 涉及大量 DOM 与数据的映射，是前端“硬核”项目
